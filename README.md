## 🧩 Core Features

### 🪄 Sorting Hat System
Interactive quiz that analyzes user personality and assigns a Hogwarts house.

### ⚔️ Spell Casting System
Click-based spell interaction with visual particle effects.

### 🏰 Houses Showcase
Dynamic cards for Gryffindor, Slytherin, Ravenclaw, Hufflepuff.

### 🧙 Character Gallery
Interactive profiles with modal-based detailed views.

### 🎧 Audio Experience
Ambient magical soundscape using Web Audio API.
